ods-button

The HTML button element represents a clickable button, which can be used in forms or anywhere in a document that needs simple, standard button functionality.

Narrow Medium Wide Max

Element <ods-button>

class OdsButton extends LitElement

Styling:

There are no styling options for this element.

Responsive state:

The ods-button element will respond without context settings as expected. Within context of another component, set context flag to true and ods-button will adapt to contextual shape.

Properties:

string: string

Value entered as string will be displayed as string value within the button element.

context: boolean = false

Context defines responsiveness of element. Set to true, element will always be 100% and respond to parent context shape.

type: string = primary

Type of button defines the visual styling. Option: secondary.

disabled: boolean = false

If set to true button will become disabled and not allow for interactions.

active: boolean = false

If set to true button will appear in active state.
