ods-button

The HTML button element represents a clickable button, which can be used in forms or anywhere in a document that needs simple, standard button functionality.

Element <ods-button>

class OdsButton extends LitElement

Styling:

There are no styling options for this element.

Responsive state:

The ods-button element will respond without context settings as expected. Within context of another component, set context flag to true and ods-button will adapt to contextual shape.
